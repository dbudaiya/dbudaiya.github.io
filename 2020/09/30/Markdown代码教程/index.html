欢迎使用 Markdown

马克飞翔是一款印象笔记打。高亮代码造的 Markdown

  • 功能丰富：支持代码高亮， aTex ;
  • 得心应手：简洁高效的编辑器，提供[桌面客户端][1]以及[l 离线 Chrome app][2]，支持移动端端 Web；
  • 深度整合：支持选中笔记本和标签，支持从印象笔记跳转编辑，轻松管理

Markdown 简介

Markdonw 是一种轻量级的标记性语言，允人们使用简单的标记文档。 —— 维基百科
