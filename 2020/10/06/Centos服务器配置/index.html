Centos 服务器相关配置

Linux 配置 YUM 源

1. 概述

CentOS（Community Enterprise Operating System）是 Linux 发行版之一，它是来自于 Red Hat Enterprise Linux（RHEL）依照开放源代码规定发布的源代码所编译而成。

2. 阿里 YUM 源

打开 Centos 的 YUM 源文件夹

cd /etc/yum.repo.d/

用 wget 下载 repo 文件

wget http://mirrors.aliyun.com/repo/Centos-7.repo

如果 wget 命令不生效，说明还没有安装 wget 工具，输入 yum -y install wget 回车进行安装。
当前目录是/etc/yum.repos.d/，刚刚下载的 Centos-7.repo 也在这个目录上

备份系统原来的 repo 文件

mv CentOs-Base.repo CentOs-Base.repo.bak

即是重命名 CentOs-Base.repo -> CentOs-Base.repo.bak

替换系统原理的 repo 文件

mv Centos-7.repo CentOs-Base.repo

即是重命名 Centos-7.repo -> CentOs-Base.repo

执行 yum 源更新命令

yum clean all
yum makecache
yum update

依次执行上述三条命令即配置完毕。

3. 本地 YUM 源

挂载光盘

mount /dev/sr0 /mnt/

进入 YUM 源配置目录

cd /etc/yum.repos.d/

备份文件夹

mkdir /etc/yum.repos.d/bak

移动原有的配置文件到备份文件夹

mv /etc/yum.repos.d/Cent* /etc/yum.repos.d/bak/

编辑本地 YUM 源文件

vim /etc/yum.repos.d/local.repo

三、Linux 软件包管理

传统的网络服务程序，FTP、POP、telnet 本质上都是不安全的，因为它们在网络上通过明文传送口令和数据，这些 数据非常容易被截获。SSH 叫做 Secure Shell 。通过 SSH，可以把传输数据进行加密，预防攻击，传输的数据 进行了压缩，可以加快传输速度。

Centos6.8 配置 JAVA 环境

查看 Linux 系统自带的 JDK

rpm -qa |grep java

有的话，进行删除

yum remove -y Java-1.7.8-openjdk

rpm 安装 JDK

rpm -ivh jdk-7u80-linux-x64.rpm

检查是否安装成功

java -version

配置环境变量,保存并退出

vi /etc/profile

执行 source 使其生效

source /etc/profile

检查是否生效

echo $PATH

在系统PATH里面有java等字眼就表示生效

测试

Centos6.8 中安装 Python

在 Linux 下安装 Python3 环境

1.源代码编译安装() 1.下载 python3 的源代码 2.解压缩源代码 3.进入源代码目录，开始编译安装 4.配置环境变量

准备编译环境

yum install zlib-devel bzip2-devel openssl-devel ncurses-devel -y

下载 Python3.6 代码包

wget https://www.python.org/ftp/python/3.6.3/Python-3.6.3.tar.xz

配置压缩环境

yum install -y xz

解压 Python3.6 代码包

tar xf Python-3.6.3.tar.xz -C /usr/local/src/

进入解压后的目录

cd /usr/local/src/Python-3.6.3/

安装编译所需依赖

软连接

ln -s /usr/local/python3/bin/python3.6 /usr/local/bin/python3

安装 Python 包

requests 包

pip install requests

paramiko 包

pip install paramiko

pumysql 包

pip install pymysql

配置系统默认 Python 版本为 3.6.3

修改旧版本

mv /usr/bin/python /usr/bin/python_bak

创建新的软连接

ln -s /usr/local/python3/bin/python3 /usr/bin/python

检查 Python 的版本

python -V

设置检查默认是否为python-3.6.3

在 Linux 配置 MYSQL 环境

SQL 服务

检查 mysql 客户端

rpm -qa | grep mysql

安装 mysql 客户端

yum install -y mysql

安装 mysql 服务器

yum install -y mysql-service

查看所有服务

cd /etc/init.d/

临时开启服务

service mysqld start

永久自启(随系统启动)

chkconfig mysqld on

查看系统自启服务

chkconfig –list

查看 mysql 网络接口

netstat -ntpl | grep mysql

修改 mysql 配置文件(修改端口号)

下面是 mysql 服务器配置文件,添加编码格式为 UTF-8

vim /etc/my.cnf

重启 SQL 服务

service mysqld restart

SQL 登录

初始化数据库设置

初始化数据库的安全设置

mysql_secure_installation
YYYNY

设置一个 root 用户允许远程的其他机器登录

grant all on . to root@’%’ identified by ‘123’

永久关闭防火墙

chkconfig iptables off

开启 SQL 服务

net start MYSQL57

登录本地数据库

mysql -hlocalhost -P3306 -uroot -p
Enter password: <输入新设的密码 newpassword>

修改 mysqladmin 错误解决方法

停止 mysql 服务

/etc/init.d/mysql stop

SQL 全局模式

mysqld_safe –user=mysql –skip-grant-tables –skip-networking &

进入数据库

mysql -u root mysql

更改密码

mysql> UPDATE user SET Password=PASSWORD(‘newpassword’) where USER=’root’ and host=’root’ or host=’localhost’;

把空的用户密码都修改成非空的密码就行了。

退出更改

mysql> FLUSH PRIVILEGES;
mysql> quit

重启 mysql 服务

/etc/init.d/mysqld restart

登录 mysql 数据库

mysql -uroot -p
Enter password: <输入新设的密码 newpassword>
